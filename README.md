# Racing Rush

¡Bienvenido a Racing Rush! Un emocionante juego de carreras arcade desarrollado con Jetpack Compose y Material 3.

## Características
- **Identidad Renovada:** Nombre y paquetes personalizados.
- **Sistema de Economía:** Recoge monedas y desbloquea nuevos coches en el Garaje.
- **Power-ups:** Escudo, Multiplicador x2 y Turbo (con invencibilidad).
- **Misiones y Logros:** Objetivos diarios y hitos globales con recompensas.
- **Anuncios:** Preparado para AdMob (Intersticial y Reward Doubler).
- **Controles:** Swipe, Tap y Acelerómetro.
